/start - команда после ввода которой показывается кнопка "начать" при нажатии на которую открывается .. Так скажем мини-лекция по теме "Загрязнение Природы и Мусорные Отходы" (чтение из файла sas.yml)

/random - команда при вводе которой пользователю скидывают рандомный сайт из файла sus.txt (сайты можно добавлять в этом нет ничего трудного) . Это нужно для дополнительного ознакомления с глобальной темой загрязнения природы и о том как с этим бороться.
